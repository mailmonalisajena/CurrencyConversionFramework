# Currency Conversion Automation Framework

This project is a Playwright-based automation framework developed in Java to validate currency conversion functionality across different currencies. 
It supports data-driven testing using CSV files and generates detailed execution reports with another CSV file by adding updated values.

## Features
- Playwright with Java for browser automation
- Data-driven testing using CSV
- Page Object Model (POM) design pattern
- Maven for dependency management
- Parallel test execution
- Logging and reporting
- Configurable environment settings

## Tech Stack
- Java
- Playwright
- Maven
- OpenCSV

## Prerequisites
- Java JDK 11 or higher
- Maven 3.6+
- Git

## Installation
1. Clone the repository:
   git clone https://github.com/your-username/CurrencyConversionFramework.git

2. Navigate to the project directory:
   cd CurrencyConversionFramework

3. Install dependencies:
   mvn clean install
